## 2024-04-24 - Improve Semantic Navigation and Focus States in H5 Games Menu
**Learning:** Custom links in `games` project lack keyboard focus indicators (browser default outlines are often insufficient or masked by `backdrop-filter` or `box-shadow`), making keyboard navigation difficult for screen readers and keyboard users. Furthermore, a list of game links should be semantically grouped inside a `<nav>` element with an `aria-label` in Chinese (e.g., `aria-label="ТИИТѕЈУЈютЇЋ"`) instead of a generic `<div>` to improve screen reader experience.
**Action:** Replace `<div>` menu containers with `<nav aria-label="ТИИТѕЈУЈютЇЋ">` and add explicit `&:focus-visible` styling (`outline: 3px solid #0984e3; outline-offset: 2px;`) to `.linkButton` in CSS Modules to ensure accessible keyboard navigation.
