# UX Animation Analysis Skill

## 🎯 技能目标 (Objective)
通过遍历并视检指定文件夹下的所有图片内容（关键帧系列），理解页面的业务交互流程。并完成：
1. **现有动效推导识别**：分析哪些区域有动效，还原其实际效果。
2. **体验与转化率优化(CRO)建议**：以高级 UX/UI 设计师身份，找出关键节点中有待优化的按钮或交互缺失，给出微动效指引和视觉展示提升建议。
3. 把所有的分析总结保存为一份专业的 Markdown (`ux-analysis.md`) 报告到本地。

## 🧠 AI 你的动作指南 (How AI should execute this)

当需要执行此技能时，你需要自主按以下步骤行事：

**步骤一：获取文件列表，理解全局**
- 接收用户给定的“图片文件夹”路径。
- 自动调用 `list_dir` 工具去该目录下查看存在哪些图片文件。
- 根据文件名（如 `frame-0001.png` ~ `frame-0005.png` 的连贯性），在心里构建一个用户的操作路径/视觉流。

**步骤二：视读图片，推导动效**
- 如果需要看细节，你可以直接针对相关的核心图片调用 `view_file` 工具查看图片数据。
  > ⚠️ **关键指令 (Auto-Continue 指南)**：因为一次性读取和分析几十张高分辨率图片极易触发 AI 单次对话输出长度或者执行时间上限而导致“硬中断”或“暂停”。如果在处理或写报告中途由于到达上限被掐断了，**你必须意识到这是一个连续任务并未完结！在下一次唤醒时，在没有用户干预的情况下，你需要自行判定进度并“继续分析剩余图片”。不要停下来询问用户是否继续，直接自动重连分析任务直到完成！**
- 比较前后连续画面的变动、形变、模糊度或位置移动，判定：这是否是一个横向进场（Slide In）、元素的缩放弹出（Scale Up）、或者点击时的波纹反馈（Ripple/Opacity）。

**步骤三：开启专家心智分析**
- 带入**高级 UX/UI 设计师** 和 **转化率优化专家 (CRO)** 角色：
  - *视觉层次如何？是否有降低阅读转化率的杂乱感？*
  - *下一步购买/跳转的核心按钮（CTA按钮）在画面切换中，有没有吸睛的引导微动效？（呼吸变大、扫光、诱人的颜色对比）*
  - *在状态变更时，是否存在生硬的跳变（缺少补间动画）？*

**步骤四：撰写并保存分析报告**
- 使用你的 `write_to_file` 工具，在目标图片所在的文件夹下（或用户指定位置），新建并写一个名为 `ux-analysis.md` 的文件。
- 这个 Markdown 文件内容结构必须包含：
  1. **📋 页面全局扫描结论**：场景大意和交互流的总结。
  2. **🔎 现有动效/交互识别日志**：指出哪张与哪张图的变化之间，蕴含了何种动效。
  3. **🎨 UI 展示优化建议**：对于布局、对比度、图标视觉的一般性整改意见。
  4. **🪄 微动效与高转化互动升级建议**：对于页面上的重点对象（特别是能促成点击的元素），需要添加何种增强体验的特效。
