# 第25课：招财猫 (Waving Pet)

我们来做一个不知疲倦的小宠物，它会一直跟你招手，除非你摸摸它的头安慰它。

## 🎯 目标
结合简单的逻辑判断和动作循环。

## 🛠 硬件清单
*   Arduino Uno 主板
*   舵机
*   触摸传感器模块 (Item 67, 通常标有 Touch)
    *   *如果没有触摸模块，用普通按键也可以替代。*

## 🔌 接线说明
*   **舵机**: Pin 9, 5V, GND
*   **触摸模块**:
    *   VCC -> 5V
    *   GND -> GND
    *   SIG (信号) -> Pin 2

## 🧩 玩耍指南 (给小朋友)
1.  **观察**：看，这个小机器人在不停地摇手，“你好！你好！”。
2.  **互动**：去摸一下那个触摸片（头顶）。
3.  **现象**：它停下来了，好像很享受你的抚摸。
4.  **离开**：手一拿开，它又开始招手了。

## 💻 代码解析 (给大人)

### 非阻塞思考
目前的 `waveHand` 函数用了 `delay`，这意味着挥手过程中（约0.6秒）它是感觉不到你摸它的。
如果你想让反应更灵敏（一摸瞬间停），需要把 `waveHand` 拆解，或者在 delay 期间也去检查 `digitalRead`。不过对于简单的玩具，目前的逻辑足够了。

### 触摸模块
触摸模块利用人体电容原理。隔着薄薄的纸或塑料也能感应到，所以你可以把它藏在玩偶的头皮下面，做一个真正的“交互玩偶”。
