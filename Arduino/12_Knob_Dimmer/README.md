# 第12课：旋钮台灯 (Knob Dimmer)

除了按钮，我们还有更有趣的输入设备——旋钮（电位器）。它可以给我们连续的控制感。

## 🎯 目标
学习 `analogRead` 读取模拟信号，以及 `map` 函数进行数值转换。

## 🛠 硬件清单
*   Arduino Uno 主板
*   旋钮电位器 B10K (清单序号 9)
*   LED 灯

## 🔌 接线说明

| 组件引脚 | 动作 | Arduino 接口 | 说明 |
| :--- | :---: | :--- | :--- |
| **电位器** 左脚 | -> | **5V** | 电源 |
| **电位器** 右脚 | -> | **GND** | 地 |
| **电位器** 中间脚 | -> | **A0** | 信号输出 (模拟) |
| **LED** 长脚 | -> | **Pin 9** | PWM 输出 |
| **LED** 短脚 | -> | **GND** | 地 |

*(电位器左右脚互换只会改变旋转方向，不影响功能)*

## 🧩 玩耍指南 (给小朋友)
1.  **动作**：慢慢转动那个像收音机音量键一样的旋钮。
2.  **现象**：看，灯光随着你的手转动，慢慢变亮，又慢慢变暗。
3.  **任务**：“请帮我把灯调到最亮！” “请把它调到一点点亮。”

## 💻 代码解析 (给大人)

### 核心语法
1.  **`analogRead(pin)`**:
    *   读取模拟电压。Arduino Uno 有 10 位精度，所以返回值是 **0 到 1023** (对应 0V 到 5V)。
2.  **`map(value, fromLow, fromHigh, toLow, toHigh)`**:
    *   这是一个超级好用的数学函数。
    *   它的意思是：把 `value` 从 `0-1023` 的比例尺，换算到 `0-255` 的比例尺。
    *   如果不转换，直接把 1023 给 `analogWrite`，它会乱套（因为它只接收0-255）。

### 常见问题
*   **抖动**: 模拟读数可能会有一点点跳动，这是正常的。
