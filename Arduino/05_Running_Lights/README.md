# 第5课：炫酷流水灯 (Running Lights)

你见过商场门口招牌上的跑马灯吗？今天我们就来做一个微缩版。

## 🎯 目标
进一步熟练掌握 `for` 循环，控制一组 LED。

## 🛠 硬件清单
*   Arduino Uno 主板
*   面包板
*   LED 灯 x 5 (颜色随意)
*   220欧姆电阻 x 5
*   杜邦线 若干

## 🔌 接线说明
我们需要接 5 个灯，分别连到 9, 10, 11, 12, 13 号口。

| 组件引脚 | 动作 | Arduino 接口 |
| :--- | :---: | :--- |
| **LED 1** 长脚 | -> | **Pin 9** |
| **LED 2** 长脚 | -> | **Pin 10** |
| **LED 3** 长脚 | -> | **Pin 11** |
| **LED 4** 长脚 | -> | **Pin 12** |
| **LED 5** 长脚 | -> | **Pin 13** |
| **所有** 短脚 | -> | **GND** |

## 🧩 玩耍指南 (给小朋友)
1.  **观察**：看光点是不是像小虫子一样从左跑到右，又跑回来？
2.  **比赛**：手指跟着光点移动，看谁跟得准。

## 💻 代码解析 (给大人)

### 核心语法
1.  **循环变量 `i`**:
    *   `for (int i = 9; i <= 13; i++)`
    *   这里的 `i` 既代表循环的次数，也正好代表了我们连接的引脚号 (9, 10, 11, 12, 13)。
    *   `digitalWrite(i, HIGH)` 这句话太妙了，它随着循环自动变成了“点亮9”、“点亮10”...
2.  **往返效果**:
    *   第二个循环 `i--` 表示数字递减，让光点倒着跑回来。

### 挑战一下
1.  **拖尾效果**：把 `digitalWrite(i, LOW)` 放到 `delay` 之后稍晚一点执行，或者不做熄灭动作，做成“进度条”填满的效果。
2.  **两头汇合**：两边的灯往中间跑？（这需要稍微复杂的逻辑，可以作为思考题）
