# 第4课：SOS 求救信号 (Functions)

如果在荒岛上，你需要用手电筒发出求救信号，该怎么做？国际通用的 SOS 信号是：三短、三长、三短。

## 🎯 目标
学习“函数”的概念。如果不使用函数，写 SOS 的代码会非常重复啰嗦。

## 🛠 硬件清单
*   Arduino Uno 主板
*   (可选) 外接一个更亮的 LED 到 Pin 13

## 🔌 接线说明
直接使用板载 **Pin 13** LED，或者参考第1课外接一个 LED。

## 🧩 玩耍指南 (给小朋友)
1.  **故事**：“小船在大海里迷路了，它要发信号给岸边的人看。”
2.  **观察**：“嘀嘀嘀 (快闪) —— 哒哒哒 (慢闪) —— 嘀嘀嘀 (快闪)”。
3.  **互动**：配合声音，当灯短闪时喊“滴”，长闪时喊“答”。

## 💻 代码解析 (给大人)

### 核心语法
1.  **自定义函数 (Function)**:
    *   我们在代码下方看到了 `void dot() { ... }` 和 `void dash() { ... }`。
    *   这就是把“短闪”和“长闪”的动作打包起来，起个名字。
    *   在 `loop` 主程序里，只要叫名字 `dot()`，Arduino 就会去执行包里的动作。
2.  **代码复用**:
    *   如果没有函数，写 3 个 S 和 3 个 O，我们需要写 9 次开关灯代码，总共 18 行 `digitalWrite`！用函数不仅代码短，而且逻辑清晰。

### 挑战一下
设计一个你自己的“暗号”。比如“两长一短”代表“我饿了”。
