# 第15课：大嗓门 (Sound Visualize)

你能看见声音吗？今天我们用灯光来“看见”声音。

## 🎯 目标
学习声音传感器（麦克风）的使用，处理波动信号。

## 🛠 硬件清单
*   Arduino Uno 主板
*   声音传感器 (清单序号 6，通常有个红色的咪头)
*   LED 灯

## 🔌 接线说明

| 组件引脚 | 动作 | Arduino 接口 | 说明 |
| :--- | :---: | :--- | :--- |
| **VCC** | -> | **5V** | |
| **GND** | -> | **GND** | |
| **AO (模拟输出)** | -> | **A0** | 必须接模拟口 |

*注意：有些声音模块只有 DO (数字输出)，只能检测“有声音/没声音”。请确认你的模块有 AO 引脚。清单里的是“声音传感器”，通常带 AO。*

## 🧩 玩耍指南 (给小朋友)
1.  **动作**：对着传感器轻轻吹气。
2.  **现象**：灯光随着气流闪烁。
3.  **游戏**：大家一起喊“啊——”，看谁能让灯变得最亮！
4.  **比喻**：这是“能量收集器”，你的声音越大，能量光束就越强。

## 💻 代码解析 (给大人)

### 声音信号处理
1.  **波形**: 声音是震动波。传感器输出的电压会在一个基准值（比如 2.5V, 读数500左右）上下波动。
2.  **`abs()`**: 我们关心的是“波动幅度”，不管它是向上波还是向下波。所以用 `abs(value - 500)` 取绝对差值。
3.  **灵敏度**: 麦克风输出的波动通常很小，所以我们需要在代码里把它“放大” (`* 5`)，才能驱动 LED 亮到 255。

### 硬件调试
如果灯一直亮或者一直不亮，请旋转传感器模块上的蓝色小电位器（灵敏度调节），直到在安静环境下 LED 微微发光或刚好熄灭。
