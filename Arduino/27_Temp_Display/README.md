# 第27课：气象站 (Temperature Monitor)

家里热吗？干吗？问问 Arduino 就知道了。

## 🎯 目标
学习使用第三方库 (Library) 驱动复杂的传感器 (DHT11)。

## 🛠 硬件清单
*   Arduino Uno 主板
*   DHT11 温湿度传感器 (Item 17, 蓝色栅格状的小盒子)

## 🔌 接线说明
DHT11 模块通常有 3 个脚：

| DHT11 引脚 | 动作 | Arduino 接口 |
| :--- | :---: | :--- |
| **VCC / +** | -> | **5V** |
| **GND / -** | -> | **GND** |
| **DATA / S** | -> | **Pin 2** |

## 🧩 玩耍指南 (给小朋友)
1.  **观察**：看电脑屏幕上的数字（比如 25°C）。
2.  **互动**：对着蓝色传感器哈一口热气。
3.  **变化**：看！湿度变高了（哈气有水份），温度也升高了。
4.  **对比**：把它拿到冰箱旁边试试？

## 💻 代码解析 (给大人)

### 库 (Library) 的概念
*   DHT11 的通信协议很复杂（有时序要求）。如果自己写代码去读高低电平会非常难。
*   通常我们直接用别人写好的库。
*   **安装库**: 在 Arduino IDE 软件里，点击 `工具` -> `管理库`，搜索 `DHT`，安装 `DHT sensor library by Adafruit`。

### `isnan()`
*   这是检查 "Is Not a Number"（是不是数字）。如果传感器接触不良或坏了，会返回 NaN，我们需要判断这种情况避免程序出错。
