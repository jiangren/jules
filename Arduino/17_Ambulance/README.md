# 第17课：紧急救援 (Ambulance Siren)

模仿救护车或防空警报的声音，这种声音听起来很急促，因为它在不断变化。

## 🎯 目标
结合 `for` 循环和 `tone` 函数，产生“滑音”效果。

## 🛠 硬件清单
*   Arduino Uno 主板
*   无源蜂鸣器
*   (可选) 红色 LED 和 蓝色 LED (结合第3课做声光报警)

## 🔌 接线说明
**蜂鸣器 -> Pin 8**

## 🧩 玩耍指南 (给小朋友)
1.  **听**：声音是“呜~~~~(变高) 呜~~~~(变低)”。
2.  **联想**：这像不像消防车或者救护车的声音？
3.  **配合**：配合第3课的警灯，就是一个完整的玩具警车了！

## 💻 代码解析 (给大人)

### 声音合成原理
*   真正的警报声是模拟信号，频率是平滑过渡的。
*   我们用 `for` 循环，每次频率增加 10Hz，看起来是断续的，但只要 `delay` 时间够短（比如 10ms），人耳听起来就是连续滑动的声音。
*   `tone(buzzerPin, i)`: 在循环中，变量 `i` 既是计数器，也是当前的频率值。

### 挑战一下
试着改变 `delay(10)` 变成 `delay(5)`，声音会变快还是变慢？（变快，像发疯的救护车）。
