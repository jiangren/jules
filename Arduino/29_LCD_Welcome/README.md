# 第29课：小小告示牌 (LCD Welcome)

LED 灯只能表达“亮”和“灭”，想说话怎么办？用屏幕！

## 🎯 目标
学习使用 LCD1602 显示屏 (I2C版本)。

## 🛠 硬件清单
*   Arduino Uno 主板
*   LCD1602 显示屏 (Item 50)
*   IIC1602 转接板 (Item 71)
    *   *注意：清单里分别列出了显示屏和转接板。如果它们没焊在一起，你需要先把转接板焊到显示屏背后。不过通常套装里是焊好的成品。确认一下屏幕背后有没有个黑色的小模块，只有4根针引出。*

## 🔌 接线说明 (I2C 接口)
I2C 只需要两根信号线，非常省引脚。

| LCD I2C 引脚 | Arduino 接口 |
| :--- | :--- |
| **GND** | **GND** |
| **VCC** | **5V** |
| **SDA** | **A4** (或者板子上标 SDA 的口) |
| **SCL** | **A5** (或者板子上标 SCL 的口) |

## 🧩 玩耍指南 (给小朋友)
1.  **惊喜**：把代码里的文字改成小朋友的名字（比如 "Hello Tom"）。
2.  **展示**：一通电，屏幕亮起蓝光，显现出他的名字。
3.  **互动**：可以结合前面的传感器，做一个“距离显示器”或者“温度显示器”。

## 💻 代码解析 (给大人)

### I2C 地址
*   代码里 `0x27` 是设备的地址。有的厂家芯片是 `0x3F`。
*   如果屏幕亮了但不显示字，或者只有方块，试着改一下这个地址。
*   或者调整背后转接板上的**蓝色电位器**（对比度调节），有时候是因为对比度太低看不见字。
