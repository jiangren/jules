# 第3课：超级警车 (Police Strobe)

这次我们要让灯光变得紧张刺激起来！模拟警车或救护车的爆闪灯效果。

## 🎯 目标
学习 `for` 循环的基本用法，制造快速闪烁的视觉效果。

## 🛠 硬件清单
*   Arduino Uno 主板
*   面包板
*   LED 灯 x 2 (红色、绿色/白色/蓝色 均可)
*   220欧姆电阻 x 2
*   杜邦线

## 🔌 接线说明

| 组件引脚 | 动作 | Arduino 接口 | 说明 |
| :--- | :---: | :--- | :--- |
| **红灯** 长脚 (+) | -> | **Pin 12** | 警灯左边 |
| **绿灯** 长脚 (+) | -> | **Pin 13** | 警灯右边 |
| **两灯** 短脚 (-) | -> | **GND** | 记得串联电阻哦 |

## 🧩 玩耍指南 (给小朋友)
1.  **场景**：把面包板放在玩具警车顶上。
2.  **故事**：“呜~呜~ 警察叔叔要去抓坏蛋了！看警灯闪得好快！”
3.  **观察**：是不是一边闪三下，然后另一边闪三下？

## 💻 代码解析 (给大人)

### 核心语法
1.  **`for` 循环**:
    ```cpp
    for(int i=0; i<3; i++) {
      // 这里的代码会重复执行 3 次
    }
    ```
    *   这就好比告诉 Arduino：“把下面这套动作（亮灯-灭灯）做3遍”。
    *   如果没有循环，我们就得把 `digitalWrite` 和 `delay` 写 6 次，代码会很长很乱。

2.  **视觉暂留**:
    *   当 `delay(50)` (0.05秒) 这样短的时间时，人眼会感觉灯光非常急促，产生“爆闪”的紧张感。

### 挑战一下
