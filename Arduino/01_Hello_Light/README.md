# 第1课：你好，灯光 (Hello Light)

这是 Arduino 编程的第一步！就像学习任何编程语言都要先打印 "Hello World" 一样，在硬件世界里，我们的 "Hello World" 就是让一盏灯闪烁起来。

## 🎯 目标
学会让 Arduino 板载的 LED 灯一闪一闪。

## 🛠 硬件清单
*   Arduino Uno 主板
*   USB 数据线

*(本节课不需要额外的传感器，直接使用主板上自带的 "L" 灯)*

## 🔌 接线说明
**本节课无需接线！**
Arduino Uno 主板的 **13号引脚 (Pin 13)** 默认连接了一个板载的小 LED 灯（通常标记为 "L"）。我们直接控制它。

## 🧩 玩耍指南 (给小朋友)
1.  **观察**：把板子插上电脑，看！板子上有一个小黄灯（或橙灯）开始像心跳一样闪动了。
2.  **互动**：爸爸/妈妈可以修改代码里的 `delay(1000)` 改成 `delay(100)`，重新上传。
3.  **问**：“看！现在它闪得是不是比刚才快了？像不像小兔子的心跳？”

## 💻 代码解析 (给大人)

### 核心语法
1.  **`void setup() { ... }`**: 初始化函数。就像准备工作，通电后**只做一次**。比如告诉 Arduino：“我要用 13 号口控制灯”。
2.  **`void loop() { ... }`**: 循环函数。这里的代码会**一遍又一遍**地执行，直到断电。
3.  **`pinMode(13, OUTPUT)`**: 设置引脚模式。`OUTPUT` 表示我们要向外“输出”电压（点灯）。
4.  **`digitalWrite(13, HIGH/LOW)`**:
    *   `HIGH`: 输出高电平（通电），灯亮。
    *   `LOW`: 输出低电平（断电），灯灭。
5.  **`delay(1000)`**: 延时。单位是毫秒。`1000` 毫秒 = 1 秒。

### 挑战一下
试着把两个 `delay` 的时间改得不一样，比如亮 100 毫秒，灭 1000 毫秒，看看会发生什么效果？（像不像闪光灯？）
