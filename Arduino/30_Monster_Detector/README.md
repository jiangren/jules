# 第30课：终极怪兽探测器 (Monster Detector)

恭喜你！这是最后一个项目。我们将把之前学到的武功结合起来，做一个全能的“防御系统”。

## 🎯 目标
综合应用：数字输出、PWM、声音、舵机、超声波传感器。

## 🛠 硬件清单
*   Arduino Uno 主板
*   超声波传感器
*   舵机
*   无源蜂鸣器
*   红色 LED, 绿色 LED

## 🔌 接线说明
这是一个大工程，接线要仔细哦！

| 组件 | 引脚 | 连到 Arduino |
| :--- | :--- | :--- |
| **超声波** | Trig | **Pin 12** |
| | Echo | **Pin 11** |
| **舵机** | Signal | **Pin 9** |
| **蜂鸣器** | + | **Pin 8** |
| **红灯** | + | **Pin 6** |
| **绿灯** | + | **Pin 5** |
| **电源** | 5V/GND | 面包板共用 |

## 🧩 玩耍指南 (给小朋友)
1.  **部署**：把这个装置放在门口。平时绿灯亮，表示“安全”。
2.  **入侵**：爸爸假装是怪兽，慢慢靠近。
3.  **触发**：一旦进入警戒范围（30厘米），装置突然“发疯”：红灯狂闪，警报大作，舵机臂（激光剑）挥舞起来！
4.  **撤退**：怪兽被吓跑了，装置恢复平静（绿灯亮）。
