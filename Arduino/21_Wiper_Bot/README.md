# 第21课：雨刷机器人 (Servo Sweep)

这是我们第一次让物体真正“动”起来！舵机（Servo）是机器人手臂关节的核心部件。

## 🎯 目标
学习使用 Arduino 标准库 `Servo.h` 控制舵机角度。

## 🛠 硬件清单
*   Arduino Uno 主板
*   舵机 (清单序号 48，P0090/SG90，通常是蓝色小电机)
*   杜邦线

## 🔌 接线说明
舵机有三根线，颜色很重要：

| 线色 | 含义 | Arduino 接口 | 说明 |
| :--- | :--- | :--- | :--- |
| **棕色/黑色** | GND | **GND** | 负极 |
| **红色** | VCC | **5V** | 正极 (一定要接5V) |
| **橙色/黄色** | Signal | **Pin 9** | 信号线 |

## 🧩 玩耍指南 (给小朋友)
1.  **观察**：看那个白色的小翅膀（舵机臂），它在左摇右摆。
2.  **联想**：
    *   “像不像汽车挡风玻璃上的雨刷？”
    *   “像不像小狗在摇尾巴？”
3.  **改装**：用胶带粘一张纸条在舵机臂上，画一个小人，它就在跟你招手了！

## 💻 代码解析 (给大人)

### 核心语法
1.  **`#include <Servo.h>`**:
    *   Arduino 自带了很多好用的工具包（库）。这句话就是把“控制舵机”的工具包拿过来用。
    *   不写这一行，Arduino 就不认识 `Servo` 这个词。
2.  **对象 (Object)**:
    *   `Servo myServo;`
    *   我们要控制一个舵机，就得给它起个名字叫 `myServo`。如果你有两个舵机，可以叫 `headServo`, `armServo`。
3.  **`myServo.write(angle)`**:
    *   直接命令舵机转到指定角度（0到180度）。
    *   舵机会尽量快地转过去。如果你想让它转慢点，就像代码里那样，把大动作拆成 1 度 1 度的小动作，中间加 `delay`。

### 注意
*   舵机耗电量较大。如果是大型项目接多个舵机，通常需要独立电源供电。只接这一个小舵机，Arduino 的 5V 还能撑得住。
