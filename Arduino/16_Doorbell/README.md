# 第16课：电子门铃 (Doorbell)

我们不仅能控制光，还能控制声音！让 Arduino 变成一个门铃。

## 🎯 目标
学习 `tone()` 和 `noTone()` 函数，控制蜂鸣器的频率。

## 🛠 硬件清单
*   Arduino Uno 主板
*   无源蜂鸣器 (清单序号 7，Passive Buzzer)
    *   *注意：请使用无源蜂鸣器。如果是有源蜂鸣器（Active），它只能发出单调的“滴”声，无法改变音调变出“叮咚”声。通常无源蜂鸣器底部能看到电路板，有源蜂鸣器底部是黑胶封死的。*
*   按键 x 1

## 🔌 接线说明

| 组件引脚 | 动作 | Arduino 接口 |
| :--- | :---: | :--- |
| **蜂鸣器** 长脚 (+) | -> | **Pin 8** |
| **蜂鸣器** 短脚 (-) | -> | **GND** |
| **按键** | -> | **Pin 2 & GND** |

## 🧩 玩耍指南 (给小朋友)
1.  **场景**：我是小客人，我要去你家做客啦！
2.  **动作**：按下门铃。
3.  **声音**：“叮——咚——”。
4.  **互动**：“请进请进！”

## 💻 代码解析 (给大人)

### 核心语法
1.  **`tone(pin, frequency)`**:
    *   让指定引脚产生特定频率的方波。
    *   `frequency`: 频率，单位赫兹 (Hz)。数值越大，声音越尖锐；数值越小，声音越低沉。
    *   `tone(8, 1000)`: 在 8 号口发出 1000Hz 的声音。
2.  **`noTone(pin)`**:
    *   停止发声。**这一点很重要**，如果不写，它会一直叫下去。

### 硬件排查
如果你的蜂鸣器发出的声音很奇怪（比如嘶哑的咔咔声），或者不管频率设多少声音都一样，那你可能拿到的是**有源蜂鸣器**。请更换一个试试。
