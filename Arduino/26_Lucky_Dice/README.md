# 第26课：幸运骰子 (Lucky Dice)

谁来洗碗？谁先走棋？让电子骰子来决定吧！

## 🎯 目标
学习控制 7段数码管 (7-Segment Display) 显示数字。

## 🛠 硬件清单
*   Arduino Uno 主板
*   1位数码管 (Item 16)
*   按键 x 1
*   220欧姆电阻 x 1 (串在公共端) 或 x 7 (串在每一段)

## 🔌 接线说明
数码管背后有 10 根针脚。
*   **上下中间的脚**: 公共端 (Common)。如果是共阴极，接 GND；共阳极接 5V。我们假设是**共阴极** (接 GND)。
*   **其他脚**: 对应 a, b, c, d, e, f, g, dp 段。需要查阅数码管引脚图，或自己用电线一根根试（亮了就记下来）。

| 数码管段 | Arduino 接口 |
| :--- | :--- |
| **A** | **Pin 6** |
| **B** | **Pin 7** |
| **C** | **Pin 8** |
| **D** | **Pin 9** |
| **E** | **Pin 10** |
| **F** | **Pin 11** |
| **G** | **Pin 12** |
| **公共端** | **GND** (串电阻) |

*注意：这可能是接线最麻烦的一课，耐心点！如果发现数字显示反相（该亮的不亮），说明你是共阳极数码管，把公共端接 5V，代码里 1 改 0，0 改 1。*

## 🧩 玩耍指南 (给小朋友)
1.  **动作**：按住按钮。
2.  **现象**：数字飞快地变来变去，像跑马灯一样。
3.  **动作**：松手！
4.  **结果**：停在一个数字上。是 6！你赢了！

## 💻 代码解析 (给大人)

### 二维数组
*   我们用 `byte numbers[10][7]` 存储了 0-9 每个数字的字形。
*   比如 `numbers[1]` 是 `{0,1,1,0,0,0,0}`，代表只点亮 B 和 C 段，这就组成了一个 "1"。
*   这比写几十个 `if/else` 要优雅得多。
