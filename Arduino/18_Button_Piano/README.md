# 第18课：小小钢琴家 (Simple Piano)

我们可以自己做一个简单的乐器，按下不同的键，发出不同的音符。

## 🎯 目标
学习 `else if` 多重条件判断，制作多按键交互装置。

## 🛠 硬件清单
*   Arduino Uno 主板
*   无源蜂鸣器
*   按键 x 3 (越多越好，如果你有更多空间)

## 🔌 接线说明
*   **蜂鸣器**: Pin 8
*   **按键 1 (Do)**: Pin 2
*   **按键 2 (Re)**: Pin 3
*   **按键 3 (Mi)**: Pin 4
*   所有按键另一端接地 (GND)。

## 🧩 玩耍指南 (给小朋友)
1.  **演奏**：你可以弹奏简单的曲子了！比如《玛丽有只小羊羔》(Mi-Re-Do-Re-Mi-Mi-Mi)。
2.  **创造**：乱按一通，创作属于你的“先锋音乐”。

## 💻 代码解析 (给大人)

### 核心语法
1.  **`else if`**:
    *   如果有多个选择，我们用 `if ... else if ... else ...` 的结构。
    *   逻辑：先看按钮1按没？按了就响Do。没按？那看按钮2按没？... 如果都没按，就闭嘴 (`noTone`)。
    *   **优先级**：这种写法有一个特点，如果同时按下按钮1和按钮2，只有按钮1会响（因为它是第一个判断的）。这叫“单音”键盘。

### 扩展
你可以查阅 `pitches.h` (Arduino 示例里常有)，找到 Fa, Sol, La, Si 的频率，把面包板插满按钮，做一个 8 音阶钢琴！
