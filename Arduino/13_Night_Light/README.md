# 第13课：聪明的小夜灯 (Night Light)

为什么路灯到了晚上自己会亮？因为它们有“眼睛”——光敏电阻。

## 🎯 目标
学习光敏传感器的工作原理，以及使用 `Serial` 串口监视器调试代码。

## 🛠 硬件清单
*   Arduino Uno 主板
*   光敏电阻传感器 (清单序号 5)
    *   *注意：如果是只有两根脚的光敏电阻元件，需要配合一个 10K 电阻搭电路。如果是 3 脚或 4 脚的模块，直接接线即可。*
    *   *这里假设使用 3脚/4脚 模块，或者搭建分压电路。清单里写的是“光敏电阻传感器”，通常指模块。如果是纯电阻，请按下面“纯电阻接法”接线。*
*   LED 灯

## 🔌 接线说明 (假设是3脚模块)

| 组件引脚 | 动作 | Arduino 接口 | 说明 |
| :--- | :---: | :--- | :--- |
| **VCC** | -> | **5V** | |
| **GND** | -> | **GND** | |
| **AO (模拟输出)** | -> | **A0** | 必须接模拟口 |

**如果是纯光敏电阻元件 (两只脚)**:
1.  光敏电阻一脚 -> 5V
2.  光敏电阻另一脚 -> A0
3.  同时，从 A0 接一个 10K 电阻 -> GND (下拉电阻)
*(这叫分压电路，没有这个电阻读数不准)*

## 🧩 玩耍指南 (给小朋友)
1.  **场景**：现在是“白天”（房间开灯）。
2.  **动作**：用手捂住那个传感器，就像太阳下山天黑了一样。
3.  **现象**：小夜灯啪地一下亮了！
4.  **动作**：松开手，天亮了，灯灭了。

## 💻 代码解析 (给大人)

### 核心技巧：串口调试
1.  **`Serial.begin(9600)`**: 告诉 Arduino 建立和电脑的通话管道。
2.  **`Serial.println(value)`**: 把读到的数字发给电脑。
3.  **查看方法**: 在 Arduino 编程软件右上角点击“放大镜”图标（串口监视器）。你会看到一串数字在跳动。
    *   观察没遮挡时的数字（比如 200）。
    *   观察遮挡时的数字（比如 900）。
    *   选一个中间值（比如 600）作为代码里的判断标准 (`if (value > 600)`)。
