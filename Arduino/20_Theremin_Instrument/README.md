# 第20课：光感特雷门琴 (Theremin)

特雷门琴 (Theremin) 是世界上最早的电子乐器之一，也是唯一不需要身体接触就能演奏的乐器。我们用光敏电阻来模拟它。

## 🎯 目标
综合运用模拟输入 (`analogRead`) 和 声音输出 (`tone`)，创造交互式体验。

## 🛠 硬件清单
*   Arduino Uno 主板
*   无源蜂鸣器
*   光敏电阻

## 🔌 接线说明
*   **光敏电阻**: A0 (参考第13课接法)
*   **蜂鸣器**: Pin 8

## 🧩 玩耍指南 (给小朋友)
1.  **动作**：把手在传感器上方上下挥舞。
2.  **现象**：
    *   手靠近（挡光） -> 声音变低沉（像牛叫）。
    *   手拿开（亮） -> 声音变尖锐（像鸟叫）。
3.  **表演**：像魔术师一样，隔空操控声音！

## 💻 代码解析 (给大人)

### 原理
*   我们将**光照强度**直接转化为了**声音频率**。
*   `map` 函数再次发挥作用：把光的变化范围 (比如 100~900) 映射到声音的可听范围 (200~800Hz)。
*   因为 `loop` 跑得很快，声音频率变化非常连贯，听起来就像滑音一样。

### 优化
如果环境光太亮或太暗，声音可能会很刺耳或者没变化。你需要结合串口监视器，看一眼 `sensorValue` 的实际范围，然后修改代码里 `map` 的参数。
