# 第22课：小小驾驶员 (Knob Steering)

把旋钮变成方向盘，精准控制机器人的动作。

## 🎯 目标
结合模拟输入 (`analogRead`) 和 舵机输出，实现“遥控”效果。

## 🛠 硬件清单
*   Arduino Uno 主板
*   舵机
*   旋钮电位器

## 🔌 接线说明
*   **舵机**: Pin 9 (参考上一课)
*   **电位器**: A0 (参考第12课)

## 🧩 玩耍指南 (给小朋友)
1.  **搭建**：把电位器想象成汽车方向盘，把舵机想象成汽车的前轮。
2.  **动作**：我想让轮子往左转，就转旋钮。
3.  **反馈**：你的手转多少，舵机就跟着转多少。这叫“同步”。

## 💻 代码解析 (给大人)

### 映射的重要性
*   电位器给出的数字是 0~1023。
*   舵机能接受的数字是 0~180。
*   `map(val, 0, 1023, 0, 180)` 就像翻译官，把“电位器语”翻译成“舵机语”。
*   如果不转换直接把 1023 传给舵机，舵机不知道该转哪儿去（通常会卡在最大角度不动）。
