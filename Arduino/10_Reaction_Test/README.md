# 第10课：谁是闪电侠 (Reaction Game)

这是我们第一个真正意义上的“游戏”。测试你的反应速度！

## 🎯 目标
学习 `random()` 随机数，以及 `while` 等待循环。

## 🛠 硬件清单
*   Arduino Uno 主板
*   大轻触按键 x 1
*   LED 灯 x 1 (建议用外接的大灯，更明显)

## 🔌 接线说明
*   **按键**: Pin 2 和 GND
*   **LED**: Pin 13 和 GND

## 🧩 玩耍指南 (给小朋友)
1.  **规则**：看着那盏灯。它现在是黑的。
2.  **等待**：不要动... 不要动... (大概等几秒)
3.  **出击**：灯亮的一瞬间，马上拍按钮！
4.  **反馈**：如果你拍到了，灯会闪两下表扬你。

## 💻 代码解析 (给大人)

### 核心语法
1.  **`random(min, max)`**:
    *   生成一个随机数。这里用来产生随机的等待时间，让游戏不可预测。
2.  **`while (条件) { ... }`**:
    *   “当...的时候，一直做...”。
    *   `while (digitalRead(buttonPin) == HIGH)`: 只要你没按按钮，我就一直卡在这里不往下走。这是最简单的“阻塞式等待”。

### 改进思路
目前的版本只是简单的流程控制。如果要真正比谁快，可以记录 `millis()` 时间戳，算出毫秒级的反应时间，并通过串口打印出来（但这属于进阶内容了）。
