# 第24课：感应垃圾桶 (Smart Trashcan)

这可是现在的网红产品——智能感应垃圾桶。手一挥，盖子自动开。

## 🎯 目标
综合运用超声波传感器测距和舵机控制。

## 🛠 硬件清单
*   Arduino Uno 主板
*   超声波传感器模块 (Item 54, HC-SR04, 像两个大眼睛)
*   舵机
*   (可选) 纸杯和硬纸板做一个垃圾桶模型

## 🔌 接线说明

| 组件 | 引脚 | 动作 | Arduino 接口 |
| :--- | :--- | :---: | :--- |
| **超声波** | VCC | -> | **5V** |
| | GND | -> | **GND** |
| | Trig (触发) | -> | **Pin 12** |
| | Echo (回声) | -> | **Pin 11** |
| **舵机** | 橙/黄 | -> | **Pin 9** |
| | 红 | -> | **5V** |
| | 棕/黑 | -> | **GND** |

## 🧩 玩耍指南 (给小朋友)
1.  **准备**：把舵机臂粘在纸板做的盖子上。
2.  **动作**：假装手里有垃圾，把手伸向“眼睛”上方。
3.  **现象**：啪！盖子开了。扔进去。
4.  **等待**：过一会儿，盖子自己关上了。真听话！

## 💻 代码解析 (给大人)

### 超声波测距原理 (HC-SR04)
1.  **喊一声**: 给 `Trig` 引脚一个高电平脉冲，模块就发出超声波（人听不见）。
2.  **听回音**: 模块等待声波撞墙反弹回来。
3.  **计时**: `pulseIn(echoPin, HIGH)` 帮助我们记录回波用了多长时间回来。
4.  **算术**: 距离 = 时间 × 速度 ÷ 2。

### 调试技巧
如果盖子乱跳：
1.  检查超声波是否对着天花板或者空旷的地方（距离太大可能读数不稳）。
2.  舵机电源是否不足（如果乱抖，尝试给舵机单独供电）。
