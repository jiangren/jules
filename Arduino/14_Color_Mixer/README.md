# 第14课：魔法调色盘 (RGB Color Mixer)

光的三原色是红、绿、蓝。只要把它们混合，就能变出世界上所有的颜色！

## 🎯 目标
控制 RGB 全彩 LED，理解颜色混合原理。

## 🛠 硬件清单
*   Arduino Uno 主板
*   10mm RGB三色灯 (清单序号 24)
    *   *注意：RGB灯有4根脚。最长的那根通常是 GND (共阴) 或 VCC (共阳)。*
*   220欧姆电阻 x 3

## 🔌 接线说明
假设是 **共阴极** (Common Cathode) RGB 灯：

| 组件引脚 | 动作 | Arduino 接口 | 说明 |
| :--- | :---: | :--- | :--- |
| **最长的脚** | -> | **GND** | 公共端 |
| **另外三脚** | -> | **Pin 9, 10, 11** | 分别接红、绿、蓝 (建议串联电阻) |

*如果不亮或者颜色不对，试着把最长的脚接 5V (共阳极)，并且修改代码逻辑。*

## 🧩 玩耍指南 (给小朋友)
1.  **观察**：看这颗灯珠，它会变魔术！
2.  **猜颜色**：现在是红色... 变成绿色了... 哎呀，红色加绿色变成了什么？是黄色！
3.  **找不同**：让小朋友观察红蓝混合出的紫色，和画画时的紫色一样吗？

## 💻 代码解析 (给大人)

### 核心原理
1.  **RGB 模型**:
    *   通过调整 R (Red), G (Green), B (Blue) 三个通道的亮度 (PWM 0-255)，可以混合出 1600 万种颜色。
    *   `analogWrite(red, 255); analogWrite(green, 255);` -> 强红 + 强绿 = 黄。
2.  **共阴 vs 共阳**:
    *   **共阴**: 公共端接 GND。给高电平(255)亮。
    *   **共阳**: 公共端接 5V。给低电平(0)亮。如果发现 `setColor(255,0,0)` 实际上亮的是青色(绿+蓝)，那就是共阳极，代码里的数值要反过来写 `255 - r`。

把上一课的电位器拿来，能不能做三个旋钮，分别控制 R、G、B 的分量？那样你就有了一个手动调色台！
