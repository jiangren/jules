# 第28课：电子尺 (Distance Ruler)

不用卷尺，我们用声音来量长度！

## 🎯 目标
深入理解超声波测距原理，并在串口输出精确数值。

## 🛠 硬件清单
*   Arduino Uno 主板
*   超声波传感器 (HC-SR04)

## 🔌 接线说明
*   **VCC** -> 5V
*   **GND** -> GND
*   **Trig** -> Pin 12
*   **Echo** -> Pin 11

## 🧩 玩耍指南 (给小朋友)
1.  **准备**：把 Arduino 连着电脑，打开串口监视器。
2.  **测量**：拿一本书，放在传感器前面慢慢移动。
3.  **读数**：屏幕上显示的数字就是距离。
4.  **游戏**：“猜猜这个有多远？”先猜，再量，看谁猜得准。

## 💻 代码解析 (给大人)
*(本课代码逻辑与第24课相同，但去掉了舵机部分，专注于数据观测)*

### 精度
*   超声波测距的精度通常在 0.5cm - 1cm 左右。
*   如果遇到软的东西（棉被、衣服），声波会被吸收，测不准。
*   如果角度太斜，声波弹飞了回不来，也测不到。
