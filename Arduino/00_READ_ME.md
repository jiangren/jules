# Arduino 学习之旅：符号与接线指南

欢迎开始您的 Arduino 学习之旅！在这个系列教程中，我们将通过 30 个有趣的实验，带您和您的孩子一起探索电子与编程的世界。

为了确保您能顺利完成实验，我们在每个教程的“接线说明”部分使用了一些符号。请先阅读下表，了解它们的含义。

## 符号说明表

| 符号/术语 | 含义 | 举例 |
| :--- | :--- | :--- |
| **->** | **连接到**。表示需要用杜邦线将左边的组件引脚连接到右边的 Arduino 接口。 | `LED长脚 -> Pin 13` (将LED的长脚连到13号口) |
| **Pin X** | **数字引脚**。Arduino 开发板上标有数字 (0-13) 的接口。 | `Pin 13` (开发板上的数字接口 13) |
| **A0 - A5** | **模拟引脚**。Arduino 开发板上标有 "ANALOG IN" 的区域。 | `A0` (开发板上的模拟输入接口 0) |
| **GND** | **接地 (负极)**。Arduino 开发板上标有 "GND" 的接口，代表电源负极。 | `GND` (开发板上的任意 GND 接口) |
| **5V / 3.3V** | **电源正极**。Arduino 开发板上标有 "5V" 或 "3.3V" 的接口。 | `5V` (开发板上的 5伏 电源输出) |
| **VCC** | **组件电源正极**。通常在传感器或模块上，需要连到 Arduino 的 5V 或 3.3V。 | `VCC -> 5V` |

## 常见组件引脚识别

*   **LED (发光二极管)**:
    *   **长脚**: 正极 (+)，通常连接到数字引脚 (Pin X)。
    *   **短脚**: 负极 (-)，必须连接到 GND。
*   **蜂鸣器**:
    *   **长脚/标有+号**: 正极，连接到数字引脚。
    *   **短脚**: 负极，连接到 GND。
*   **轻触按键**:
    *   通常是对角线导通，我们会在教程中详细说明接法。

## 给家长的建议

这些教程旨在让您先理解原理，然后引导孩子观察现象。
*   **3岁小朋友怎么玩？** 重点在于“物理反馈”。让他们负责按下按钮、遮挡光线、对着麦克风吹气，然后观察灯光、声音或电机的变化。
*   **安全第一**: 虽然 Arduino 电压很低（5V）是安全的，但请避免让孩子吞食小零件（如 LED、电阻）。

祝你们玩得开心！
