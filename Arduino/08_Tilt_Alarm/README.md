# 第8课：倾斜报警器 (Tilt Alarm)

利用重力来控制电路！这可以用来做一个“防盗宝盒”。

## 🎯 目标
认识“倾斜传感器”（滚珠开关）的原理。

## 🛠 硬件清单
*   Arduino Uno 主板
*   倾斜开关传感器 (清单序号 10)
*   有源蜂鸣器 (清单序号 8，通常贴着贴纸) 或 LED

## 🔌 接线说明

| 组件引脚 | 动作 | Arduino 接口 | 说明 |
| :--- | :---: | :--- | :--- |
| **倾斜传感器** 一端 | -> | **Pin 2** | 信号 |
| **倾斜传感器** 另一端 | -> | **GND** | 地 |
| **蜂鸣器** 长脚 (+) | -> | **Pin 8** | 报警输出 |
| **蜂鸣器** 短脚 (-) | -> | **GND** | 地 |

*(如果分不清蜂鸣器，可以用板载 LED Pin 13 代替测试)*

## 🧩 玩耍指南 (给小朋友)
1.  **制作**：把传感器粘在一个小盒子上。
2.  **游戏**：“这个盒子里有宝藏，谁要是敢把它拿歪了，警报就会响！”
3.  **尝试**：小心翼翼地平移盒子（不响），稍微一歪（滴——！）。

## 💻 代码解析 (给大人)

### 原理
*   **倾斜开关**：里面有两个金属触点和一个小金属球。
*   水平时，球连通触点（导通，LOW）。
*   倾斜时，球滚走，断开触点（断路，HIGH）。
*   代码逻辑和按钮其实一模一样，只是触发方式变了。

### 硬件注意
*   **有源 vs 无源蜂鸣器**:
    *   **有源 (Active)**: 给电就叫 (就像 LED 亮一样)。适合做简单报警。
    *   **无源 (Passive)**: 给电不叫，需要给频率 (tone)。适合做音乐。
    *   本课假设是有源蜂鸣器。如果发出的声音是“哒哒”声而不是长鸣，可能是无源的，那就像 LED 一样看光亮就行。
