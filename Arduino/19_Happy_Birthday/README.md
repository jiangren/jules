# 第19课：生日快乐歌 (Music Player)

不用手弹，让 Arduino 自动为你演奏一首曲子。

## 🎯 目标
学习数组 (Array) 的概念，存储一段旋律。

## 🛠 硬件清单
*   Arduino Uno 主板
*   无源蜂鸣器

## 🔌 接线说明
**蜂鸣器 -> Pin 8**

## 🧩 玩耍指南 (给小朋友)
1.  **场景**：今天是某人的生日（或者假装是）。
2.  **动作**：插上电源。
3.  **聆听**：Arduino 唱起了“祝你生日快乐...”。
4.  **庆祝**：大家一起拍手唱歌！

## 💻 代码解析 (给大人)

### 核心语法
1.  **数组 `int melody[] = { ... }`**:
    *   就像一个排好队的储物柜，每个格子里放一个音符。
    *   我们可以用编号（索引）来访问它们，比如 `melody[0]` 是第一个音符。
2.  **两个数组配合**:
    *   `melody[]` 存的是**音高** (唱什么)。
    *   `noteDurations[]` 存的是**节拍** (唱多久)。
    *   循环里的 `i` 同时控制这两个数组，保证音高和节拍一一对应。
