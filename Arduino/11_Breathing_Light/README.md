# 第11课：会呼吸的灯 (Breathing Light)

数字信号只有 0 (灭) 和 1 (亮)。但为什么电脑的休眠灯能“半亮半暗”呢？这就是 PWM 技术的魔力。

## 🎯 目标
学习 `analogWrite` 和 PWM 概念，控制 LED 的亮度。

## 🛠 硬件清单
*   Arduino Uno 主板
*   LED 灯 (建议绿色或蓝色，效果好)
*   220欧姆电阻

## 🔌 接线说明
**注意**：这次不能随便插了。必须插在板子上数字旁带 **~** 符号的引脚（如 3, 5, 6, 9, 10, 11）。我们选 **Pin 9**。

| 组件引脚 | 动作 | Arduino 接口 | 说明 |
| :--- | :---: | :--- | :--- |
| **LED** 长脚 (+) | -> | **Pin 9** | PWM 输出口 |
| **LED** 短脚 (-) | -> | **GND** | 地 |

## 🧩 玩耍指南 (给小朋友)
1.  **观察**：这个灯不是突然亮、突然灭的。
2.  **比喻**：“它像是在睡觉呼吸一样。吸气——变亮；呼气——变暗。”
3.  **互动**：跟着灯光的节奏一起深呼吸。

## 💻 代码解析 (给大人)

### 核心语法
1.  **`analogWrite(pin, value)`**:
    *   虽然叫“模拟写入”，但 Arduino Uno 输出的其实是 **PWM (脉宽调制)** 信号。
    *   `value`: 范围是 **0 到 255**。
    *   0 = 完全灭 (0V)。
    *   255 = 完全亮 (5V)。
    *   128 = 50% 亮度。
2.  **PWM 原理**:
    *   它其实是在极短的时间内快速开关灯。开的时间长，看起来就亮；关的时间长，看起来就暗。骗过了人的眼睛。

### 挑战一下
修改 `delay(30)` 的数值，让呼吸变得急促（跑步后）或缓慢（睡觉时）。
