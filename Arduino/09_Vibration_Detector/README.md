# 第9课：地震仪 (Vibration Detector)

谁在敲门？或者发生了地震？用这个传感器来感知震动。

## 🎯 目标
了解灵敏度较高的开关传感器。

## 🛠 硬件清单
*   Arduino Uno 主板
*   震动传感器 (清单序号 14，通常是一个黑色的圆柱体或者带有弹簧的元件)
*   LED 灯

## 🔌 接线说明
同按钮和倾斜开关：
**传感器一脚 -> Pin 2**
**传感器另一脚 -> GND**

## 🧩 玩耍指南 (给小朋友)
1.  **场景**：把装置放在桌面上。
2.  **动作**：用手指轻轻敲击桌面。
3.  **现象**：灯光闪烁！
4.  **比喻**：“这是怪兽探测器，走路声音太大的话，它就会发现你！”

## 💻 代码解析 (给大人)

### 传感器原理
*   内部通常是一根很细的弹簧。平时弹簧不动。
*   震动时，弹簧摆动碰到旁边的金属壁，瞬间导通电路。
*   因为它导通的时间极短（毫秒级），我们在代码里加了 `delay(500)`，这就是为了“捕捉”那个瞬间并延长显示，否则灯闪太快人眼看不清。
